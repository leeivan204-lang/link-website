# 網頁連結轉運站 - 使用者說明書

歡迎使用「網頁連結轉運站」管理系統。本手冊將引導您如何管理網站內容。

## 1. 系統登入
1.  開啟網站首頁。
2.  滾動至頁面最下方 (Footer)。
3.  點擊 **[ 編輯者登入 ]**。
4.  輸入密碼 (預設: `admin`) 並點擊驗證。
5.  驗證成功後，畫面會出現編輯介面。

## 2. 管理連結 (Links)
### 新增連結
1.  在「發布連結」區塊中：
    -   **網址**: 貼上目標網站 URL (必填)。
    -   **標題**: 輸入顯示的名稱 (必填)。
    -   **描述**: 簡單說明該網站用途。
    -   **圖片**: 
        -   **網址**: 貼上圖片 URL。
        -   **上傳**: 點擊「相機圖示」選擇電腦中的圖片，系統會自動壓縮。
    -   **注意事項**: 若有特殊提醒 (如: "1/20 截止") 可在此輸入。系統會自動偵測日期格式 (如 `1/20`) 並自動加入公告欄。
2.  點擊 **「發布連結」** 按鈕。

### 刪除連結
1.  登入後，每個連結卡片左上角會出現紅色的 **「×」** 按鈕。
2.  點擊並確認刪除即可。

## 3. 管理公告 (Notices)
### 新增公告
1.  在「重要事項提醒」區塊，輸入文字。
2.  點擊 **「新增」**。

### 清除公告
1.  點擊紅色的 **「清除」** 按鈕可刪除所有公告。
2.  若要刪除單一公告，點擊該公告旁的 **「×」**。

## 4. 如何讓其他人看到更新？
本系統採用「靜態部署」技術。當您在本地電腦 (使用 `python app.py`) 完成編輯後，您看到的只是**您電腦上的檔案**。

**要讓全世界看到，您必須執行「同步」：**
1.  開啟終端機 (Terminal/CMD)。
2.  輸入以下指令：
    ```bash
    git add .
    git commit -m "更新內容"
    git push
    ```
3.  等待約 2 分鐘，GitHub 會自動將您的最新內容發布到網路上。
