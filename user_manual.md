# Link Aggregation Tool User Manual | 網站連結集結工具使用說明書

## 📖 簡介
本工具旨在協助使用者整理常用連結，並提供公告與備註功能。由於移除了自動抓取功能，使用者擁有完全的控制權，可自定義標題、描述與圖片。

## 🚀 快速上手 (Quick Start)

### 1. 新增連結
1.  **貼上網址 (必填)**：將目標網頁網址貼上至「在此貼上網址...」欄位。
2.  **輸入標題 (必填)**：自行設定該網頁的標題。
3.  **輸入描述 (選填)**：輸入該網頁的簡短介紹。
4.  **輸入圖片網址 (選填)**：若希望顯示縮圖，請貼上圖片連結（需為公開可存取之網址）。
5.  **連結注意事項 (選填)**：若有特別提醒（如日期、截止日），可在此輸入。若包含日期格式（如 `1/15`），將自動加入上方公告欄。
6.  點擊 **「發布連結」**。

### 2. 編輯與刪除
- **刪除連結**：將滑鼠移至卡片上方（或點擊），右上角會出現紅色 **×** 按鈕，點擊並確認即可刪除。
- **編輯備註**：點擊卡片內部的備註區域（鉛筆圖示），即可修改備註文字。勾選「同步公告」可將修改後的備註同步更新至公告欄。

## 📢 公告欄系統
- **自動同步**：在新增連結時，若備註包含日期，系統會自動在公告欄建立一則新公告。
- **手動新增**：在公告欄下方的輸入框直接輸入文字並點擊「儲存」。
- **刪除公告**：點擊公告旁的 **×** 按鈕即可移除。
- **清除全部**：點擊「清除」按鈕可一次清空所有公告。

## 🔑 編輯者權限
- 為了防止誤刪，部分功能（刪除、編輯）需要「編輯者登入」才可操作。
- 點擊頁尾的 `[ 編輯者登入 ]`，輸入密碼後即可解鎖完整功能。

## ⚠️ 常見問題
**Q: 圖片無法顯示？**
A: 請確認您貼上的圖片網址是否有效（可以在瀏覽器新分頁開啟測試）。部分網站可能有防盜鏈機制，導致無法在外部顯示。

**Q: 資料存在哪裡？**
A: 資料儲存在雲端伺服器 (Render) 的檔案系統中。若重新部署，資料可能會保留（取決於您是否有掛載 Persistent Disk，目前版本為簡易檔案儲存）。
