# 連結集結工具 - 使用者說明書 (User Manual)

歡迎使用連結集結工具！這份指南將協助從零開始使用本軟體。

## 1. 快速入門

### 介面概覽
- **輸入框**：位於頁面中間上方，用於貼上您想要收藏的網址。
- **加入按鈕 (Add Link)**：點擊後系統會開始處理該網址。
- **連結卡片區**：位於下方，展示您已收藏的所有連結。

### 如何加入新連結
1. 複製您想要收藏的網頁網址 (例如：`https://www.youtube.com/`).
2. 在工具的「輸入框」中貼上網址。
3. **(選填)** 在下方的「注意事項」欄位輸入備註 (例如："3/10 需閱讀")。
4. 按下 **Enter** 鍵或點擊 **Add Link** 按鈕。
5. 系統會顯示 "Getting metadata..." 表示正在抓取資料。
6. 完成後，一張包含圖標、標題、簡介與**備註** (帶有 ⚠️ 符號) 的卡片就會出現在列表中。

### 智慧公告 (Smart Notice)
- 若您在「注意事項」中輸入了日期 (如 `1/15` 或 `2024-03-20`)，系統會自動將該條備註添加到頁面頂部的「重要事項提醒」欄位。
- 公告中會自動附上 **[連結]** 按鈕，點擊即可直接前往該網頁。

### 如何瀏覽連結
- 點擊任一卡片，會在**新分頁**中開啟該網站。
- 卡片上會顯示網站來源 (例如：youtube.com)，方便辨識。

## 2. 常見問題 (FAQ)

**Q: 為什麼有些網站抓不到圖片或標題？**
A: 這取決於目標網站是否提供了標準的 Open Graph (OG) 標籤。如果對方網站有防爬蟲機制或格式不標準，系統會使用預設圖片和網址作為標題。

**Q: 我關閉視窗後，連結還會存在嗎？**
A: 是的。系統會將連結資料儲存於伺服器端的 `links.json` 檔案中，下次開啟時會自動載入。

**Q: 支援手機瀏覽嗎？**
A: 是的，本工具採用響應式設計 (Responsive Design)，在手機、平板和電腦上都能完美顯示。
